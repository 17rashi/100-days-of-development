Declaration of Variable

let myName;

let myAge;

To use a variable, you've first got to create it — more accurately, we call this declaring the variable.

Initialization of variable

let myName = 'Rashi';

Variables are case sensitive — so myage is a different variable from myAge.

NOTE: variable naming rules

JavaScript is a "dynamically typed language", which means that, unlike some other languages, you don't need to specify what data type a variable will contain (numbers, strings, arrays, etc).

Constant in JavaScript

  • You must intialise constant when you delare them
  • you can't assign them a new value after you've initialised the variable.

    • When to use const and when to use let

    Use const when you can, and use let when you have to. This means that if you can initialize a variable when you declare it, and don't need to reassign it later, make it a constant.
