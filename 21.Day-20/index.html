A high-definition language

Javascript is a scripting language that allows you to implement complex features on webpage
Example: displaying timely content updates, interactive maps, animated 2D/3D graphics, scrolling video jukeboxes, etc.

API

  1. DOM(Document Object Model)
  2. Geolocation API
  3. Canvas and WebGL
  4. Audio and Video APIs

Running Order

It generally runs from top to bottom.


   const name1 = document.getElementById('name');

   name1.addEventListener('click',updateName);

   function updateName(){
      const newName=prompt('Enter a new name');
      name1.textContent=`Player 1: ${newName}`;
   }
