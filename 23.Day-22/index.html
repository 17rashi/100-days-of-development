ERRORS IN JAVASCRIPT

We generally encounter two types of error in JavaScript i.e., Logical Error and Syntax Error.

Some imp points
  • Javascript is case sensitive i.e., addeventListener and addEventListener is not same.
  • Use console log to check the error
  • Use operators properly. like, === and = is not same.
