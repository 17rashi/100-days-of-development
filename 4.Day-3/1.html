Basics of CSS

  • CSS stands for Cascading Style Sheet
  • CSS includes adding visuals like colour, fonts and layouts, etc.

To add CSS to html page we use link tag

There are three types of adding CSS to webpage

  1. Inline CSS
  2. Internal CSS
  3. External CSS

Prefrence to CSS

Inline < Internal < External

Class Selector

  1. Class Selectors
  2. Id Selectors
  3. Element/type Selectors

=> For grouping selectors

Seperate the selectors by comma's

For Example:
p, .class-name, div { CSS properties;}

=> For Nesting Selectors

Seperate the selectors like class, id by space only.

For Example:
p .class-name div {CSS properties;}

=> For Chaining Selectors

To use chaining we use the combination of selectors without any space in between them

For Example:
h1.class-name{CSS Properties;}
