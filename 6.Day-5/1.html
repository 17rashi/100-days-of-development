INLINE ELEMENTS

  • we cannot set the height and width of inline elements.
  • An inline elements does not start on a new line and only takes up as much width as required.

      • Examples:

    1. <span>
    2. <a>
    3. <img>

    BLOCK ELEMENTS

  • Block elements always starts with a new line and takes up entire width of a line.

      • Examples:

    1. <div>
    2. <h1> to <h6>
    3. <p>
    4. <form>
    5. <header>
    6. <footer> and <section>
    NOTE: Inline-Block has the property of both,
    inline and block. We can control the height and width of these elements
